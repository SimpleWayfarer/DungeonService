Сервис для создания и сбора пати для DnD, Зова Ктулуху и Starfinder.
# Архитектура
Стэк бэкенда: Python + FastAPI + gRPC
Микросервисы:
 -api_gateway для управления бэкендом
 -auth_service для авторизации и регистрации пользователей с помощью JWT токена
   +PostgreSQL для хранения данных пользователей 
   +Redis для кэширования сессии
 -party_service для создания и сбора пати 
 -character_service для создания игрового персонажа в различных ролевых системах
# Для запуска:
1) Развернуть 
```
sudo docker compose up -d
```
2) Перейти на 
```
http://localhost:3000/
